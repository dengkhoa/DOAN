53/*



54 * Bit banding



55 *



56 * The Cortex memory map includes two bit-band regions.



57 * These regions map each word in an alias region of memory to a bit in



58 * a bit-band region of memory. Writing to a word in the alias region



59 * has the same effect as a read-modify-write operation on the targeted bit



60 * in the bit-band region.



61 *



62 * In the STM32F4xxx both peripheral registers and SRAM are mapped in



63 * a bit-band region. This allows single bit-band write and read operations



64 * to be performed.



65 *



66 * A mapping formula shows how to reference each word in the alias region



67 * to a corresponding bit in the bit-band region. The mapping formula is:



68 * bit_word_addr = bit_band_base + (byte_offset x 32) + (bit_number × 4)



69 *



70 * BB_base      is the base address of the bit-band region for a register. (PERIPH_BB_BASE at 0x42000000 or SRAM_BB_BASE at 0x22000000)



71 * Reg_base     is the base address of a register in the memory region. (PERIPH_BASE at 0x40000000 or SRAM_BASE at 0x20000000)



72 * Register     is a register in the memory region that contains the targeted bit.



73 * Bit_number   is the targeted bit number (0 - 31).



74 *



75 * PERIPH_BB_BASE, SRAM_BB_BASE, PERIPH_BASE, and SRAM_BASE are defined in stm32f10x.h



76 *



77 * Example for determining bit banding bit.



78 * Determine DMA Transfer Complete Flag using bit banding.



79 * TCIFx is the flag for DMA Channel x



80 * This bit is set by hardware. It is cleared by software writing 1 to



81 * the corresponding bit in the DMA_IFCR register.



82 * TCIFx = 0: No transfer complete (TC) event on channel x



83 * TCIFx = 1: A transfer complete (TC) event occurred on channel x



84 *



85 * DMA Channel:       1 2 3  4  5  6  7



86 * Flag bit position: 1 5 9 13 17 21 25 (in DMA_ISR register)



87 * (For DMA2 Channel 6 and 7 is irrelevant since it only has 5 channels.)



88 *



89 * CTCIFx: Channel x transfer complete clear (x = 1 ..7)



90 * This bit is set and cleared by software.



91 * CTCIFx = 0: No effect



92 * CTCIFx = 1: Clear the corresponding TCIF flag in the DMA_ISR register



93 *



94 */



95#define BitBanding(Register, Bit_number, Reg_base, BB_base) ((volatile uint32_t*) ((BB_base) + (((uint32_t) &(Register)) - ((Reg_base))<<5) + ((Bit_number)<<2)))
