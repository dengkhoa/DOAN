35 * Bit banding



36 *



37 * The Cortex memory map includes two bit-band regions.



38 * These regions map each word in an alias region of memory to a bit in



39 * a bit-band region of memory. Writing to a word in the alias region



40 * has the same effect as a read-modify-write operation on the targeted bit



41 * in the bit-band region.



42 *



43 * In the STM32F4xxx both peripheral registers and SRAM are mapped in



44 * a bit-band region. This allows single bit-band write and read operations



45 * to be performed.



46 *



47 * A mapping formula shows how to reference each word in the alias region



48 * to a corresponding bit in the bit-band region. The mapping formula is:



49 * bit_word_addr = bit_band_base + (byte_offset x 32) + (bit_number × 4)



50 *



51 * BB_base      is the base address of the bit-band region for a register. (PERIPH_BB_BASE at 0x42000000 or SRAM_BB_BASE at 0x22000000)



52 * Reg_base     is the base address of a register in the memory region. (PERIPH_BASE at 0x40000000 or SRAM_BASE at 0x20000000)



53 * Register     is a register in the memory region that contains the targeted bit.



54 * Bit_number   is the targeted bit number (0 - 31).



55 *



56 * PERIPH_BB_BASE, SRAM_BB_BASE, PERIPH_BASE, and SRAM_BASE are defined in stm32f10x.h



57 *



58 * Example for determining bit banding bit.



59 * Determine DMA Transfer Complete Flag using bit banding.



60 * TCIFx is the flag for DMA Channel x



61 * This bit is set by hardware. It is cleared by software writing 1 to



62 * the corresponding bit in the DMA_IFCR register.



63 * TCIFx = 0: No transfer complete (TC) event on channel x



64 * TCIFx = 1: A transfer complete (TC) event occurred on channel x



65 *



66 * DMA Channel:       1 2 3  4  5  6  7



67 * Flag bit position: 1 5 9 13 17 21 25 (in DMA_ISR register)



68 * (For DMA2 Channel 6 and 7 is irrelevant since it only has 5 channels.)



69 *



70 * CTCIFx: Channel x transfer complete clear (x = 1 ..7)



71 * This bit is set and cleared by software.



72 * CTCIFx = 0: No effect



73 * CTCIFx = 1: Clear the corresponding TCIF flag in the DMA_ISR register
